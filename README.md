# ExeMaker

this is a batch project that convert any file (python,batch,vbs...) to .exe

what to do?

step 1. put ExeMaker.bat to your desktop

step 2. put your project to ExeMaker.bat

step 3. wait for a few seconds

step 4. enjoy! :)
